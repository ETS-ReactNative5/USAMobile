# USAMobile App

The mobile version

Prerequisite 
1. Install `Node/NPM`
2. Install `Yarn`
3. Install Xcode, If you have mac
4. On Window Install Android Studio

Follow steps to complete setup
https://reactnative.dev/docs/environment-setup

Run following commands to run the project 

`yarn install`

Run IOS
`yarn run ios`

Run Android 
`yarn run android`
